# Nomflix

Learning React and ES6 by building a Movie Discovery App.

## Screens

- Home
- TV shows
- Search
- Detail

## API Verbs

- Now playing (Movie)
- Top Rated (Movie)
- Popular (TV, Movie)
- Upcoming (TV, Movie)
- Airing Today (TV)
- TV Show Detail
- Movie Detail
- Search (Movie, TV)
